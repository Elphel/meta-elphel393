# meta-elphel393

This layer provides official support for Elphel Inc. reconfigurable cameras.

## Compatibility

Compatible with Kubuntu 20.04 (which uses glibc 2.31)

## Supported boards/machines
Camera boards to be supported by this layer:

* **Elphel NC393**
* **MicroZed**
* **Zedboard** (not tested)
* **ZC706** (not tested)
